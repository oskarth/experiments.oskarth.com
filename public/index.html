shell

Sat, Jun 20, 2015 What is a shell? A shell is just another computer program. It is the main user interface to operating systems that are similar to Unix. An operating system is responsible for having several programs run on one computer, and also to tries to abstract away (to draw away attention from) the specific hardware the computer is running on, so the same program can run on many different types of computers. Unix is a special type of operating system that was developed at Bell Labs (a special place where a lot of important computer things happened) by programmers (people who make computers do what they want) like Ken Thompson, Dennis Ritchie, and Brian Kernighan.
