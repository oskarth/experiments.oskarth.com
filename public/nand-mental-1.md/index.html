In the last few weeks I've been following the excellent Nand To Tetris
course. It builds up a computer from scratch. The main reason I took
the course is because I wanted to have a better sense of how a
computer works, and roughly how one could build one. In a sense its
value is proportional to how well it serves as a mental value for the
real world.

So how does it stack up? This is the first of several parts series,
starting with the first three chapters of Elements of Computer Systems
book, where I write a bunch of assertions of how things work, and then
diff these with what I've found out in the real world. Let's begin.

From Chapter 1.

1. All logic functions can be built up from a NAND gate.

2. The above is how we build computers, make logic gates out of more
   elementary logic gates.

3. There are a few exceptions to (2) above. Physically, it's a
question of cost, speed and performance.
